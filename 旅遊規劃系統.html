      🖨️ 列印 / 另存 PDF

      將總覽、預算、行程、清單一次輸出成 PDF（瀏覽器列印 → 另存為 PDF）。

      📅 行事曆 .ics

      下載後雙擊匯入 Google / Apple / Outlook 行事曆，每日行程自動建立事件（含交通建議）。

      📊 試算表 CSV

      含預算明細、行程、清單，可用 Excel / Google 試算表開啟。

      📋 複製行程文字（LINE 分享）

      把逐日行程＋預算摘要變成漂亮的文字訊息，直接貼到 LINE 群組給親朋好友。

      💾 備份 / 還原

      JSON 完整備份，可換電腦或分享給同行旅伴匯入。

      👨‍👩‍👧 怎麼分享給親朋好友？

      • 傳整個系統：把「旅遊規劃系統.html」這個檔案用 LINE／Email 傳給對方，對方存到手機或電腦後用瀏覽器打開即可使用（免安裝、離線可用）。
      • 傳你的行程：①「複製行程文字」貼到 LINE 群組最快；②「備份」匯出 JSON 給旅伴，對方按「還原」就能看到一模一樣的行程和清單。
      • 同步到大家的行事曆：下載 .ics 丟到群組，每個人匯入自己的 Google／Apple 行事曆；或由一人匯入 Google 日曆後用「與特定使用者共用」邀請全家。
      • 列印給長輩：「列印／存 PDF」輸出紙本，總覽＋行程＋清單一目了然。
