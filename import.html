CSV 格式說明

系統支援以下欄位名稱（CSV 標題行）：
學號 或 ID — 學號（必填）
姓名 — 中文姓名
英文名 — 英文姓名
等級 或 班級 — 班級名稱
來源 或 國籍 — 國籍
性別 — 男/女
電話 — 聯絡電話
棟 — 宿舍棟別（如 12F）
房號 — 房間號碼
床 — 床號
狀態 — 在住/工場實習
生日 — 出生日期

⚠ 注意：請勿在 CSV 中包含身分證號或居留證號欄位，以保護個人資料。
系統將自動為每位學生產生獨立的亂數識別碼作為 QR Code 內容。
